◆ arrumarVetor()

void arrumarVetor ( DISTANCIAS *  distancias,
int  nEtapas 
)

Esta função organiza o vetor distancias.

Esta função coloca primeiramente a etapa partida na primeira posição do vetor ou seja a posição 0 para depois organizar o restante vetor.

Para organizar o restante vetor é retirado de cada etapa iniciada por 'E' o caracter 'E' para assim organizar por numeros de etapa

Foi necessário criar variáveis auxiliares para guardar os valores contidos numa das posições do vetor, neste caso do i pois os valores de i vão ser substituidos pelos de j perdendo assim os valores

Parameters
auxDist➔ variavel auxiliar para guardar a distancia
auxEtapaI➔ variavel auxiliar para guardar o nome da etapa inicial
auxEtapaF➔ variavel auxiliar para guardar o nome da etapa inicial
